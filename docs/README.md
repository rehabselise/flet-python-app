# IMIS x ISDP Team 8 - Mood Tracker Documentation

## Project Overview

Mindful Momdents is a mood tracking application designed to help users log their daily emotions and activities.

## Features

- **Mood Logging**: Track your mood on a 1-10 scale with notes
- **Daily Mood Entry**: Quick mood logging from the home screen
- **Activity Tracking**: Log predefined activities (exercise, studying, socializing, etc.)
